# PROJECT MANAGER CLI

## Estado del proyecto

En desarrollo activo.

## Descripción

Es un programa de CLI para administrar proyectos de forma facil y rapida desde la consola.

## Cómo ejecutar
1. Tener instalado python
2. Clonar el repo:
    ````
    git clone https://github.com/0plimplim0/projectManager-CLI
    ```
3. Ejecutar:
    ```
    python ./main.py -h
    ```

## Features básicas
- Crear proyectos.
- Listar proyectos.
- Crear tareas dentro de los proyectos.
- Marcar tareas como completadas (done).
- Ver estado de un proyecto.
- Persistencia local con sqlite.
